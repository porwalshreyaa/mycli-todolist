# 'mycli' TodoList

### Commands:

1. Add a Todo
    
    ```
    mycli add <todo> 
    ```

2. Update Todo by todo-id
    ```
    mycli update <todoid> <todo> 
    ```

3. Mark task as "done"
    ```
    mycli mark-as-done <todoid> 
    ```

4. Mark task "in-progress"
    ```
    mycli mark-in-progress <todoid> 
    ```

5. Delete Todo by id
    ```
    mycli delete <todoid> 
    ```
